Java版本

Nov 20, 2022
337字

省流：java se6= jdk6+jre6

历史原因

jdk1.6= jdk6

jdk1.8= jdk8

以此类推，现在全都直接叫jdk+版本了，没有1.x、

如现在Java SE 19，对应JDK19

Java SE 8，对应JDK8，或者叫1.8

维基百科可以直接查看：https://zh.wikipedia.org/wiki/Java%E7%89%88%E6%9C%AC%E6%AD%B7%E5%8F%B2

另外是三个类型，可以了解一下：

  • JavaSE（Java Platform，Standard Edition）Java标准版
  • JavaME（Java Platform，Micro Edition）Java微型版
  • JavaEE（Java Platform，Enterprise Edition) JAVA企业版

参考：https://www.jb51.net/article/214196.htm
